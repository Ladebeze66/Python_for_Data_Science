# ft_package

A sample test package created for 42 School Python Piscine.

## Description

This package provides simple utilities for list manipulation, specifically counting occurrences of elements in lists.

## Installation

You can install this package using either of the following commands:

pip install ./dist/ft_package-0.0.1.tar.gz
# or
pip install ./dist/ft_package-0.0.1-py3-none-any.whl
